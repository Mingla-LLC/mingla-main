## Overview

### What Are Experience Types?

**Experience Types** define the **social context** and **relationship dynamic** for an outing. They answer the question: **"Who are you going with and what's the vibe?"**

Think of them as the **intent layer** that shapes the entire experience.

### What Are Categories?

**Categories** define the **actual activity** or **type of venue**. They answer the question: **"What are you actually doing?"**

Think of them as the **activity layer** that describes the experience itself.

### How They Work Together

```
User Selects:
├─► Experience Type(s) → WHO you're with & VIBE
│   Example: "First Date" → Intimate, conversation-friendly
│
└─► Categories → WHAT you're doing
    Example: "Sip & Chill" → Coffee shop, wine bar
    
Result:
Cards shown = Coffee shops optimized for first dates
(quiet, intimate, good conversation spaces)
```

### The Two-Layer System

```
┌────────────────────────────────────────────────────┐
│         LAYER 1: EXPERIENCE TYPE                   │
│   (Social Context - Filters Categories)            │
│                                                    │
│  Solo  │ First  │ Romantic │ Friendly │ Group │ Biz│
│ Adventure Date  │          │         │  Fun  │    │
└──────────┬─────────────────────────────────────────┘
           │ Filters ↓
┌──────────▼─────────────────────────────────────────┐
│         LAYER 2: CATEGORIES                        │
│   (Activity Type - Shows Specific Cards)           │
│                                                    │
│  Stroll │ Sip & │ Casual │ Screen │ Creative │ ... │
│         │ Chill │  Eats  │ Relax  │         │     │
└──────────┬─────────────────────────────────────────┘
           │ Shows ↓
┌──────────▼─────────────────────────────────────────┐
│              CARDS DISPLAYED                       │
│   Only cards matching BOTH filters                 │
└────────────────────────────────────────────────────┘
```

## Experience Types (6 Types)

### 1. Solo Adventure 👤

**Definition**: Independent exploration and self-discovery

**Who It's For**:
- Solo travelers
- People who enjoy alone time
- Those exploring new hobbies independently
- Self-care focused individuals

**Vibe Characteristics**:
- No pressure to engage socially
- Freedom to move at own pace
- Introspective and reflective
- Flexible timing
- Personal growth oriented

**Compatible Categories**: **ALL 10 categories**
- Stroll ✅
- Sip & Chill ✅
- Casual Eats ✅
- Screen & Relax ✅
- Creative & Hands-On ✅
- Picnics ✅
- Play & Move ✅
- Dining Experiences ✅
- Wellness Dates ✅
- Freestyle ✅

**Card Characteristics When Selected**:
- Work-friendly spaces (WiFi, power outlets)
- Solo-friendly seating (bar seating, communal tables)
- No reservation required
- Quiet or moderately ambient
- Easy to navigate alone
- Self-guided experiences

**Example Cards**:
- Coffee shop with ample solo seating
- Museum with audio guide
- Scenic walking trail with map
- Yoga class drop-in
- Solo-friendly restaurant with bar seating

### 2. First Date 💕

**Definition**: Initial romantic meeting with someone new

**Who It's For**:
- People meeting romantic interests for the first time
- Early dating phase (1st-3rd dates)
- Exploratory romantic connections
- Building rapport and chemistry

**Vibe Characteristics**:
- Conversation-friendly (not too loud)
- Casual yet thoughtful
- Low-pressure atmosphere
- Easy escape routes (not locked in)
- Impression-making without trying too hard
- 1-2 hour duration typical

**Compatible Categories**: **7 categories**
- Stroll ✅ (Easy conversation while walking)
- Sip & Chill ✅ (Classic coffee/drink date)
- Picnics ✅ (Casual outdoor bonding)
- Screen & Relax ✅ (Shared experience, less pressure)
- Creative & Hands-On ✅ (Fun ice-breaker activities)
- Play & Move ✅ (Active, playful energy)
- Dining Experiences ✅ (Upscale but not too formal)
- ❌ Casual Eats (too casual for first impression)
- ❌ Wellness Dates (too intimate for first date)
- ❌ Freestyle (too unpredictable)

**Card Characteristics When Selected**:
- Quietness score: 70-90 (conversation-friendly)
- Intimacy score: 60-80 (not too intense)
- Duration: 1-2 hours max
- Seating: Face-to-face tables, outdoor options
- Lighting: Soft, flattering
- Ambience: Warm, welcoming, not stuffy
- Price range: $15-50 per person (accessible but quality)
- Has escape routes (near transit, easy to leave)

**Example Cards**:
- Cozy coffee shop with 2-person tables
- Park stroll with scenic viewpoints + ice cream stop
- Interactive museum exhibit
- Casual wine bar with small bites
- Mini-golf with outdoor seating nearby

## Categories (10 Categories)

### 1. Take a Stroll 🚶 (ROUTE-BASED)

**Definition**: Light, scenic outings combining walking with optional food/drink stop

**What It Includes**:
- Parks and trails
- Waterfronts and beaches
- Botanical gardens
- Urban walking routes
- Scenic viewpoints
- Historic neighborhoods

**Unique Characteristics**:
- **Multi-stop experiences** (3-5 locations minimum)
- Route-based (start → waypoints → end)
- Outdoor focus
- Weather-dependent
- Free or low-cost ($0-25)
- 30-60 minute duration
- Paired with food/drink stop (café, ice cream, etc.)

**Route Structure**:
```
Start: Anchor Location (park, trail)
  ↓
Mid: Scenic points along route
  ↓
End: Paired food stop (café, bakery)
```

**Experience Type Compatibility**:
- ✅ Solo Adventure (self-paced exploration)
- ✅ First Date (easy conversation, no pressure)
- ✅ Romantic (scenic sunset walks)
- ✅ Friendly (casual catch-up)
- ❌ Group Fun (too slow for large groups)
- ✅ Business (walking meetings)

**Card Attributes**:
- `isMultiStop: true`
- `routeSteps: 3-5 locations`
- `weatherDependent: true`
- `priceRange: "$0-25"`
- `scenicPoints: []`
- `pairedFoodStop: { name, address, priceRange }`

**Example Cards**:
- Golden Gate Park loop + Blue Bottle Coffee
- Embarcadero waterfront + Ferry Building
- Botanical garden stroll + tea house
- Beach sunset walk + ice cream shop

## Matching & Scoring

### Category Match Score Component

Part of the overall match score (0-100):

```javascript
function calculateCategoryScore(card, userExperienceTypes, userCategories) {
  let score = 0;
  
  // PART 1: Direct category match (+100 base)
  if (userCategories.includes(card.category)) {
    score = 100;
  }
  
  // PART 2: Experience type compatibility boost
  const categoryExperienceMap = {
    stroll: ['soloAdventure', 'firstDate', 'romantic', 'friendly', 'business'],
    sipChill: ['soloAdventure', 'firstDate', 'romantic', 'friendly', 'business'],
    casualEats: ['soloAdventure', 'friendly', 'groupFun'],
    screenRelax: ['soloAdventure', 'firstDate', 'friendly', 'groupFun'],
    creative: ['soloAdventure', 'firstDate', 'friendly', 'groupFun'],
    picnics: ['soloAdventure', 'firstDate', 'romantic', 'friendly'],
    playMove: ['soloAdventure', 'firstDate', 'friendly', 'groupFun'],
    diningExp: ['soloAdventure', 'firstDate', 'romantic', 'friendly', 'business'],
    wellness: ['soloAdventure', 'romantic', 'friendly'],
    freestyle: ['soloAdventure', 'friendly', 'groupFun']
  };
  
  const compatibleTypes = categoryExperienceMap[card.category] || [];
  const matchingTypes = userExperienceTypes.filter(type => 
    compatibleTypes.includes(type)
  );
  
  // Boost for each matching experience type
  if (matchingTypes.length > 0) {
    score += 10 * matchingTypes.length; // +10 per match
  }
  
  return Math.min(100, score);
}
```

Different categories use different attributes:

```javascript
function applyExperienceTypeFit(card, userExperienceTypes) {
  if (!card.categoryAttributes?.experienceTypeFit) {
    return 50; // Default neutral
  }
  
  const fitScores = userExperienceTypes.map(type => 
    card.categoryAttributes.experienceTypeFit[type] || 50
  );
  
  // Average of all selected experience types
  const avgFit = fitScores.reduce((a, b) => a + b, 0) / fitScores.length;
  
  return avgFit; // 0-100
}
```

## Technical Implementation

### PreferencesSheet.tsx - Category Filtering

```typescript
// Get filtered categories based on selected experience types
const getFilteredCategories = (): Category[] => {
  // If no experience types selected, show all categories
  if (selectedExperiences.length === 0) {
    return categories;
  }

  // Define category IDs for each experience type
  const experienceTypeFilters: Record<string, string[]> = {
    soloAdventure: ['stroll', 'sipChill', 'casualEats', 'screenRelax', 'creative', 'picnics', 'playMove', 'diningExp', 'wellness', 'freestyle'],
    firstDate: ['stroll', 'sipChill', 'picnics', 'screenRelax', 'creative', 'playMove', 'diningExp'],
    romantic: ['sipChill', 'picnics', 'diningExp', 'wellness'],
    friendly: ['stroll', 'sipChill', 'casualEats', 'screenRelax', 'creative', 'picnics', 'playMove', 'diningExp', 'wellness', 'freestyle'],
    groupFun: ['playMove', 'creative', 'casualEats', 'screenRelax', 'freestyle'],
    business: ['stroll', 'sipChill', 'diningExp']
  };

  // Collect all category IDs that match ANY selected experience type (UNION)
  const relevantCategoryIds = new Set<string>();
  selectedExperiences.forEach(expType => {
    const categoryIds = experienceTypeFilters[expType] || [];
    categoryIds.forEach(catId => relevantCategoryIds.add(catId));
  });

  // Filter categories array to only show relevant ones
  return categories.filter(cat => relevantCategoryIds.has(cat.id));
};
```

### SwipeableCards.tsx - Category Filtering

```typescript
// Category filtering - filter cards based on user's selected categories
const categoryFilteredCards = useMemo(() => {

      };

    return card;
  });
}, [budgetFilteredCards, userPreferences?.categories]);
```

### Backend - Card Generation with Categories

```typescript
async function generateCardBatch(userId, preferences, batchSize = 10) {
  // Build query with category filter
  const query = {
    status: 'live',
    isDeleted: false,
    
    // Category filter
    category: { $in: preferences.categories || ALL_CATEGORIES },
    
    // Experience type compatibility
    $or: preferences.experienceTypes.map(expType => ({
      [`categoryAttributes.experienceTypeFit.${expType}`]: { $gte: 70 }
    }))
  };
  
  const cards = await Card.find(query);
  
  // Score each card
  const scoredCards = cards.map(card => {
    const categoryScore = calculateCategoryScore(
      card,
      preferences.experienceTypes,
      preferences.categories
    );
    
    return {
      ...card,
      categoryScore,
      matchScore: calculateOverallMatchScore(card, preferences)
    };
  });
  
  return scoredCards.sort((a, b) => b.matchScore - a.matchScore).slice(0, batchSize);
}
```
