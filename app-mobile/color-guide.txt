Color Usage Principles
Primary orange (#eb7825) is ONLY for primary actions and selected states
White backgrounds for all content containers to maintain clean, modern look
Semantic colors (green/red) for clear positive/negative actions
Gray scale for hierarchy: darker = more important, lighter = less important
Consistent borders using #E5E7EB for visual separation
Hover states always 10-15% darker than base color
Active/pressed states slightly darker than hover
Disabled states always use gray scale, never colored
Focus rings use 20% opacity of primary color for accessibility
